# Lingo Trainer

A German language learning app with TikTok-style vertical swipe cards. Built with React + Vite.

## Agent skills

### Issue tracker

Issues live in GitHub Issues (`r3tr0devilz/lingo-trainer`). See `docs/agents/issue-tracker.md`.

### Triage labels

Using default label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo — one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
